# shadcn/ui Components

Run this command to install all required shadcn/ui components:

```bash
npx shadcn-ui@latest init
npx shadcn-ui@latest add button input label select card badge avatar dialog alert-dialog dropdown-menu progress textarea separator tabs tooltip
```

Or install all at once:
```bash
npx shadcn-ui@latest add button input label select card badge avatar dialog alert-dialog dropdown-menu progress textarea separator tabs tooltip --yes
```
